cutgeneratingfunctionology.dff.find_epsilon_interval_continuous_dff(fn, perturb)¶

Compute the interval [minus_epsilon, plus_epsilon] such that (fn + epsilon * perturb) is superadditive for epsilon in this interval. Assumes that fn is superadditive. If one of the epsilons is 0, the function bails out early and returns 0, 0.

cutgeneratingfunctionology.dff.phi_s_delta(delta=1/10, s=3/2, inf=5)¶

Create a family of piecewise linear general DFFs, which have breakpoints: [-inf,-delta, 0, delta, 1-delta,1,1+delta,inf] and slopes: [s,2s,0,1/(1-s*delta),0,2s,s] in each affine piece. The function is maximal if delta is small and s is large.
